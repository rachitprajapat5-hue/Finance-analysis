## Overview

This project focuses on performing financial data analysis using MySQL. The objective of the project is to analyze financial records, identify business trends, evaluate key performance indicators (KPIs), and generate meaningful insights using SQL queries.

The project demonstrates practical skills in data extraction, data cleaning, aggregation, and financial reporting using structured query language (SQL).

---

## Objectives

* Analyze financial datasets to identify revenue and profit trends
* Perform data cleaning and preprocessing
* Generate business insights using SQL queries
* Calculate important financial KPIs
* Improve decision-making through data analysis

---

## Tools & Technologies

* MySQL
* SQL
* MySQL Workbench

## Dataset

The dataset contains financial transaction records including:
